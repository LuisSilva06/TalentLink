TalentLink — Conectando talentos, propósito e futuro

Resumo do Projeto
O **TalentLink** é uma aplicação web desenvolvida em **React + Vite + Tailwind CSS**, criada como uma rede profissional inspirada no LinkedIn.  
A plataforma tem como objetivo **conectar talentos e oportunidades**, exibindo perfis de profissionais com informações como área de atuação, cidade, tecnologias dominadas e experiências.

O projeto apresenta:
- Interface moderna e responsiva.
- Filtros dinâmicos por área, cidade e tecnologia.
- Tema **Dark/Light Mode** automático.
- Perfis com dados gerados de forma fictícia e personalizada.

Instalação do Projeto

Para executar o TalentLink localmente em sua máquina, siga os passos abaixo.
Certifique-se de ter o Node.js instalado (versão 18 ou superior) antes de começar.

Baixe o projeto
Faça o download do arquivo .zip do TalentLink e extraia em uma pasta de sua preferência.
Se preferir, você também pode clonar o repositório diretamente do GitHub com o comando:

git clone https://github.com/LuisSilva06/TalentLink.git

Acesse a pasta do projeto
No terminal, entre na pasta onde o projeto foi extraído:

cd talentlink

Instale as dependências
Execute o comando abaixo para instalar todas as bibliotecas necessárias:

npm install

Isso instalará automaticamente o React, Tailwind CSS, Vite e demais pacotes usados no projeto.

Inicie o servidor de desenvolvimento
Após a instalação, execute o comando:

npm run dev

Esse comando iniciará o servidor local de desenvolvimento.

Acesse o projeto no navegador
Após iniciar o servidor, o terminal exibirá um endereço semelhante a:

http://localhost:5173/

Copie e cole esse link no navegador para visualizar o TalentLink funcionando.
